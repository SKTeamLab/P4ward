Command Line Reference
======================

.. option:: -h, --help

    Show the help message and exit

.. option:: --config_file FILE

    Path to the .ini file where the user configuration is stored

.. option:: --write_default

    Copy full configuration file with default values to the current working directory and quit the program.

.. option:: --check_lig_matches

    Check that P4ward can accurately identify the ligands and linker in the protac structure.

.. option:: --benchmark

    Benchmark a modelling run.

.. option:: --ref_ligase FILE

    Path to the reference pose for the E3 ligase for benchmarking.
